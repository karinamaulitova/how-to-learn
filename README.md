# Учебный проект "Путешествие по России" для Яндекс Практикума
 
 *Данный проект должен покзать пользователям интересные идеи для путшествий по России. Проект сделан по методологии БЭМ Nested. Сетка проекта сделана с использованием flexbox и grid Layout, чтобы добавить адаптивности и кроссбраузерности проекту. Проект так же выполнен для нескольких разрешений экрана, чтобы пользователи могли зайти на него с разных устройств.*
